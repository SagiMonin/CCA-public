This is a softwate implementation of the algorithm described in:

E. Z. Psarakis, G. D. Evangelidis, "An enhanced correlation-based 
method for stereo correspondence with subpixel accuracy", 
IEEE Int. Conf. on Computer Vision (ICCV'05), 2005.

released by Georgios Evangelidis.

It is available in the following webpage:

http://xanthippi.ceid.upatras.gr/people/evangelidis/encc/
